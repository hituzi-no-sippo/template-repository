== Trim Trailing Blank Line

// tag::blank_line_example[]

.Before (`_` is Space, `>-` is Tab, `^M` is Carriage Return, `$` is last line)
----
apple

banana


cacao


donuts


^M
______
>->->-

$
----

// end::blank_line_example[]

----
apple

banana


cacao
