.Supported formats
* YAML
* JSON
* TOML

.Configuration file
* `.lefthook.<yml|json|toml>`
* `lefthook.<yml|json|toml>`

=== Local configuration

If you want to local configuration, write config to `.lefthook-local.yml`.
The local configuration will overwrite configuration file (`.lefthook.yml`).
